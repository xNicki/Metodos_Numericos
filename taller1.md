# Algoritmo 2 (Bubble Sort)

## Corrida de escritorio

Vector inicial:  

$v1 = [3, 2, 5, 8, 4, 1]$

| i   | Vector             |
| --- | -- |
| 0   | $[3, 2, 5, 8, 4, 1]$ |
| 1   | $[3, 2, 5, 4, 1, 8]$ |
| 2   | $[3, 2, 4, 1, 5, 8]$ |
| 3   | $[3, 2, 1, 4, 5, 8]$ |
| ... |                |

Resultado final
$$v_{sorted} = [1,2,3,4,5,8]$$

Casos de prueba
* $v_2 = [-1,0,4,5,6,7]$
* $v_3$ = 100_000 números aleatorios entre -200 y 145
